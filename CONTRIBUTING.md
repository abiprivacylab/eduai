# Contributing to EduAI

Thank you for your interest in contributing to **EduAI**, a privacy‑first, multi‑model AI learning platform designed for students, teachers, and global classrooms. This project is built with a strong focus on safety, transparency, and educational impact. Contributions of all kinds are welcome — from code to documentation to ideas.

---

## 🧱 Project Structure
- **frontend/** — Next.js UI for students, teachers, and school dashboards  
- **backend/** (coming soon) — FastAPI service for model routing, safety, and credits  
- **docs/** — Architecture, design notes, and long‑term vision  

---
